# Проверка переключателя "Registration Completed"

## ✅ Код присутствует в собранном файле

Проверено: строка "Registration Completed" найдена в собранном файле `dist/assets/index-*.js`

## Проблема: переключатель не виден в браузере

### Возможные причины:

1. **Docker контейнер не пересобран** - старый код в контейнере
2. **Браузер кэширует старую версию** - нужно очистить кэш
3. **Фронтенд не перезапущен** - изменения не применены

## Решение:

### Шаг 1: Пересобрать Docker контейнер

```bash
cd infra
docker-compose up -d --build frontend
```

### Шаг 2: Очистить кэш браузера

**Вариант A: Жесткая перезагрузка**
- Windows: `Ctrl + Shift + R` или `Ctrl + F5`
- Mac: `Cmd + Shift + R`

**Вариант B: Через DevTools**
1. Откройте DevTools (F12)
2. Правой кнопкой мыши на кнопку обновления
3. Выберите "Empty Cache and Hard Reload"

**Вариант C: Полная очистка кэша**
1. Откройте DevTools (F12)
2. Перейдите на вкладку "Application" (или "Хранилище")
3. Нажмите "Clear storage" → "Clear site data"

### Шаг 3: Проверить в консоли браузера

1. Откройте DevTools (F12)
2. Перейдите на вкладку "Console"
3. Проверьте, нет ли ошибок
4. Введите в консоли:
   ```javascript
   // Проверка, что код загружен
   document.querySelector('label[for*="finishedOnboarding"]')
   ```

### Шаг 4: Проверить Network tab

1. Откройте DevTools (F12)
2. Перейдите на вкладку "Network"
3. Обновите страницу (Ctrl+R)
4. Найдите файл `index-*.js`
5. Проверьте, что он загружается с сервера (Status 200, не из кэша)
6. Откройте файл и найдите "Registration Completed"

## Где должен быть переключатель:

1. Откройте админку
2. Перейдите в раздел "People"
3. Нажмите "Edit" на любом пользователе
4. В форме редактирования должны быть видны три переключателя:
   - ✅ Notifications (строка 592-603)
   - ✅ Matches (строка 605-616)
   - ✅ **Registration Completed** (строка 618-629) ← должен быть здесь

## Если все еще не видно:

1. Проверьте, что используете правильный URL админки
2. Проверьте, что фронтенд действительно пересобран: `npm run build` в `services/frontend`
3. Проверьте логи Docker контейнера: `docker-compose logs frontend`
4. Попробуйте открыть в режиме инкогнито (это отключит кэш)
